Problem: Reverse a given array.

```
Input = [1, 2, 3, 4, 5, 6], Output = [6, 5, 4, 3, 2, 1]
```

Use Python's in-built reverse method.

In Python:

```python
def reverse_array(arr):
  return arr.reverse()
```

In Javascript:

```javascript
function reverse(arr) {
  return arr.reverse();
}
```
